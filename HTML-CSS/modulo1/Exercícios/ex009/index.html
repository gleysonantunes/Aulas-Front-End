Trabalhando com listas

Listas ordenadas

Podemos usar os "type":

  • 1 = 1, 2, 3
  • A = A, B, C
  • a = a, b, c
  • I = I, II, III, IV
  • i = i, ii, iii, iv

Também podemos usar o "start" que diz em que número a lista começa

Exemplo:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a Faculdade
  6. Voltar para casa
  7. Jantar
  8. Dormir

Listas não ordenadas

Podemos usar os "type":

  • disc
  • circle
  • square

Exemplo:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
